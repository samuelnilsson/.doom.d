#+title: Config

* UI

** Theme

#+begin_src emacs-lisp
(setq doom-theme 'doom-one)
#+end_src

** Window

Enable line numbers.

#+begin_src emacs-lisp
(setq display-line-numbers-type t)
#+end_src


* Org mode

** Directory

Set directory for org files.

#+begin_src emacs-lisp
(setq org-directory "~/wiki/")
#+end_src
